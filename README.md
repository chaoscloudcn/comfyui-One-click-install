由于comfyui更新太快为了便于测试所以开发这里个一键脚本安装包

使用方法：

创建一个存放comfyui的目录：例如：d:/comfyui_2026.3.10

下载 Install_ComfyUI.bat 保存到 d:/comfyui_2026.3.10  

提前先安装好显卡驱动，保证科学上网 ，运行 Install_ComfyUI.bat 

他会开始自动下载安装，只要的网络正常，全自动安装

如果安装中途出现下载错误，重新启动还是会自动继续安装
